# 홈 리뉴얼 PRD (데크센터)

## 배경
- 현재 홈은 정보 나열 중심이라 브랜드 인상과 견적 전환에 집중도가 낮음
- 시공업자/일반 소비자 모두가 품질과 A/S 신뢰를 빠르게 판단해야 함
- 전문성을 보여줄 시공사례 강조가 필요함
- 레퍼런스: millboard.com 수준의 프리미엄 톤을 참고

## 목표
- 브랜드 신뢰와 전문성 인상 강화
- 1차 CTA인 견적 문의 흐름 단순화
- 체류시간을 늘릴 수 있는 콘텐츠 흐름 구성
- 핵심 가치(내구성/품질/A/S) 전달력을 높임

## KPI
- 평균 체류시간 증가
- 견적 문의 클릭수 증가
- 문의 전환수 증가
- 시공사례 섹션 스크롤 도달률 증가

## 타깃
- 시공업자(자재 선택과 납품 신뢰가 중요)
- 일반 소비자(디자인과 유지관리 부담이 중요)
- 상업 시설 발주자(TBD)
- 조경/리노베이션 의사결정자(TBD)

## 핵심 가치 제안
- 내구성 중심 메시지 강화
- 품질 기준과 관리 공정 제시
- A/S 신속 대응과 유지관리 안내
- 전문가 컨설팅 기반 추천 강조

## 범위
- 히어로 영상 높이 50vh 적용
- 히어로 영역 확대와 메시지 재정렬
- 제품 카테고리 요약 및 진입 경로 정리
- 시공사례 하이라이트 섹션 강화
- 비교/가이드 요약 섹션 추가 또는 개선
- 견적 CTA 재배치와 강조

## 비목표
- 견적 시스템 자체의 UX 재설계
- 전체 리브랜딩(로고/네이밍) 변경
- 다국어/글로벌 확장
- CMS 도입은 이번 범위에서 제외

## 콘텐츠 요구사항
- 히어로 영상과 핵심 카피 정리
- 시공사례 이미지 선별 및 캡션 보강
- 제품 카테고리별 짧은 가치 문구
- 신뢰 근거(수치/인증/보증) 후보 수집

## 제약/의존성
- Astro + MDX 구조 유지
- 배포 환경은 추후 결정
- GA4 등 분석 도구는 추후 추가
- 주 1회 수준의 콘텐츠 업데이트 고려(현재는 MDX 유지)

## 오픈 질문
- 업데이트 담당자 및 CMS 필요 여부
- 문의 전환의 구체적 정의(전화/폼/외부 링크)
- 신뢰 근거에 사용할 공식 데이터 확보
