# 홈 리뉴얼 TRD (데크센터)

## 기술 스택/구조
- Astro 5 정적 사이트 유지
- 콘텐츠는 `src/content` MDX 컬렉션 사용
- 전역 레이아웃은 `src/layouts/BaseLayout.astro`
- 컴포넌트는 `src/components`에서 재사용

## 데이터/콘텐츠 소스
- 제품 데이터는 `products` 컬렉션 활용
- 시공사례는 `cases` 컬렉션에서 3개 하이라이트
- 블로그는 `blog` 컬렉션으로 가이드/비교 필터
- 이미지/영상은 `public` 또는 외부 CDN 사용

## 홈 섹션 구조
- 히어로: 영상 중심, CTA 1차 배치
- 제품 카테고리: 4개 카드 그리드
- 시공사례: 3개 카드 + 상세 진입
- 비교/가이드: 블로그 필터 카드 또는 링크
- 견적 CTA: 페이지 하단 핵심 재확인

## 컴포넌트 변경/추가
- `src/components/Hero.astro` 높이 및 미디어 비중 확장
- `src/components/Card.astro` 이미지 높이/라운드/그리드 간격 조정
- `src/components/Section.astro` 타이틀/서브카피 스타일 강화
- 신규 `TrustBar` 또는 `Stats` 컴포넌트 검토
- `src/pages/index.astro` 섹션 순서와 카피 업데이트

## 스타일/타이포 방향
- 전문적이고 묵직한 톤을 유지하되 여백 확대
- 히어로 타이틀 크기와 라인하이트 재조정
- 색상은 기존 팔레트 기반으로 대비 개선
- 섹션별 배경 톤 분리로 가독성 강화

## 반응형/미디어
- 모바일에서는 히어로 미디어를 상단 배치
- 영상은 `object-fit: cover` 유지, 비율 튜닝
- 히어로 영상 높이 50vh 적용
- 768px 이하에서 그리드 1열 전환 유지
- LCP 고려해 히어로 영상은 용량 최적화

## 성능/접근성/SEO
- 히어로 영상에 `poster` 이미지 제공 검토
- 이미지에는 `loading`과 대체 텍스트 유지
- `og`/`twitter` 메타는 `BaseLayout` 기준 유지
- 키워드: 합성데크, 합성목재, 데크, WPC 데크

## 분석/추적(추후)
- GA4는 런치 이후 추가
- 이벤트: CTA 클릭, 시공사례 카드 클릭
- 스크롤 깊이 측정은 추가 도구 필요
- 사전 요구사항은 추후 정의
