데크센터 앱을 처음 사용하시나요? 이 가이드를 따라하면 누구나 쉽게 첫 번째 데크를 설계할 수 있습니다.

## 1. 회원가입 및 로그인

데크센터는 이메일 또는 소셜 계정으로 간편하게 가입할 수 있습니다.

1. [데크센터 앱](/app) 페이지로 이동합니다
2. **무료로 시작하기** 버튼을 클릭합니다
3. 이메일 주소를 입력하거나 Google/카카오 계정으로 로그인합니다
4. 이메일 인증을 완료하면 바로 설계를 시작할 수 있습니다

> 무료 플랜으로도 기본 3D 설계와 20종의 자재를 사용할 수 있습니다.

## 2. 대시보드 둘러보기

로그인하면 대시보드가 나타납니다. 주요 메뉴를 살펴보겠습니다.

| 메뉴 | 설명 |
|------|------|
| 새 프로젝트 | 새로운 데크 설계 시작 |
| 내 프로젝트 | 저장된 설계 목록 |
| 템플릿 | 미리 만들어진 디자인 시작점 |
| 자재 라이브러리 | 사용 가능한 자재 둘러보기 |

## 3. 공간 크기 입력하기

**새 프로젝트**를 클릭하면 공간 입력 화면이 나타납니다.

### 기본 정보 입력
- **가로 길이**: 데크의 너비 (미터 단위)
- **세로 길이**: 데크의 깊이 (미터 단위)
- **형태 선택**: 직사각형, L자, ㄷ자, 자유형

### 설치 위치
- 발코니
- 루프탑
- 마당
- 테라스

위치에 따라 추천 자재와 설계 옵션이 달라집니다.

## 4. 3D 뷰 조작법

설계 화면에서 3D 뷰를 자유롭게 조작할 수 있습니다.

### 마우스 조작
- **좌클릭 드래그**: 화면 회전
- **우클릭 드래그**: 화면 이동
- **스크롤**: 줌 인/아웃

### 키보드 단축키
- `R`: 뷰 리셋
- `T`: 탑뷰 (위에서 보기)
- `F`: 전체 화면 맞춤
- `Space`: 자재 패널 토글

## 5. 자재 선택하기

오른쪽 패널에서 원하는 자재를 선택합니다.

### 자재 종류
1. **목재**: 자연스러운 질감, 따뜻한 느낌
2. **합성목(WPC)**: 내구성 우수, 관리 편리
3. **알루미늄**: 모던한 디자인, 최고의 내구성

### 자재 적용 방법
1. 자재 라이브러리에서 원하는 자재 클릭
2. 3D 뷰에서 적용할 영역 클릭
3. 실시간으로 변경된 모습 확인

> 여러 자재를 조합해서 사용할 수도 있습니다. 테두리는 알루미늄, 바닥은 합성목처럼 말이죠.

## 6. 견적 확인하기

설계가 완료되면 하단의 **견적 보기** 버튼을 클릭합니다.

### 견적서에 포함되는 정보
- 총 면적 및 자재 수량
- 자재별 단가 및 합계
- 예상 시공 비용 (참고용)
- 부자재 목록

### 저장 및 공유
- **PDF 다운로드**: 견적서와 도면을 PDF로 저장
- **링크 공유**: 고유 링크로 다른 사람과 공유
- **프로젝트 저장**: 클라우드에 저장 후 나중에 수정
