## 봄 (3-5월): 시즌 오픈

### 3월: 봄맞이 대청소

**해야 할 일:**

1. **낙엽, 이물질 제거**
   - 겨울 동안 쌓인 낙엽 완전 제거
   - 데크 틈새 이물질 청소
   - 배수구 막힘 확인

2. **손상 점검**
   - 갈라짐, 뒤틀림 확인
   - 고정 나사 풀림 점검
   - 곰팡이 발생 여부 확인

3. **세척**
   - 물청소로 먼지 제거
   - 심한 오염은 중성 세제 사용
   - 고압 세척 시 1,500psi 이하

### 4월: 오일 스테인 도장 (목재)

![데크에 오일 스테인을 도장하는 모습](https://images.unsplash.com/photo-1558618666-fcd25c85cd64?auto=format&fit=crop&q=80&w=800)

**최적의 도장 조건:**
- 기온: 15-25°C
- 습도: 60% 이하
- 비 예보 없는 날
- 세척 후 48시간 건조 후

**도장 순서:**
1. 표면 샌딩 (필요시)
2. 먼지 완전 제거
3. 나무결 방향으로 균일하게 도포
4. 24시간 건조
5. 2차 도장 (권장)

### 5월: 점검 및 보수

- 도장 상태 확인
- 부분 보수 작업
- 가구 배치 시작

## 여름 (6-8월): 주의 관리

### 6월: 장마 대비

**필수 점검:**

| 점검 항목 | 확인 사항 |
|----------|----------|
| 배수구 | 막힘 없이 원활한가 |
| 배수 경사 | 물이 고이는 곳 없는가 |
| 하부 통풍 | 막힌 곳 없는가 |
| 방수 처리 | 벗겨진 곳 없는가 |

### 7월: 고온기 관리

**주의사항:**
- 직사광선으로 표면 온도 상승 (맨발 주의)
- 합성목/알루미늄은 그늘막 설치 고려
- 화분 받침 사용 (물 고임 방지)
- 바비큐 시 보호 매트 필수

### 8월: 점검

- 고온으로 인한 변형 점검
- 목재 갈라짐 확인
- 팽창 간격 확인 (좁아졌는지)

## 가을 (9-11월): 월동 준비

### 9월: 가을 대청소

**청소 항목:**
1. 낙엽 즉시 제거 (습기 유발)
2. 데크 전체 물청소
3. 곰팡이 발생 부위 처리
4. 가구 정리 및 보관

### 10월: 방부 처리 (목재)

**방부 처리 순서:**
1. 완전 건조 상태 확인
2. 기존 도장 벗겨짐 부위 샌딩
3. 방부제 도포
4. 48시간 건조

**합성목/알루미늄:**
- 물청소만으로 충분
- 심한 오염 시 중성 세제 사용

### 11월: 월동 준비

![겨울을 앞둔 데크 월동 준비](https://images.unsplash.com/photo-1600585154526-990dced4db0d?auto=format&fit=crop&q=80&w=800)

**필수 작업:**

| 항목 | 작업 내용 |
|------|----------|
| 가구 보관 | 실내 이동 또는 방수 커버 |
| 화분 정리 | 동파 방지를 위해 실내 이동 |
| 배수 점검 | 배수구 완전 개방 확인 |
| 보호 커버 | 목재는 부분 커버 고려 |
| 최종 점검 | 손상 부위 기록 (봄에 보수) |

## 겨울 (12-2월): 최소 관리

### 적설 관리

**해야 할 것:**
- 눈 쌓이면 플라스틱 삽으로 제거
- 얼음은 자연 해빙 대기
- 미끄럼 방지 모래 뿌리기 (필요시)

**하지 말아야 할 것:**
- ❌ 금속 삽 사용 (표면 손상)
- ❌ 제설제 사용 (자재 손상)
- ❌ 뜨거운 물로 얼음 제거 (균열 유발)

### 월 1회 점검

- 과도한 적설로 인한 처짐 확인
- 결빙으로 인한 균열 확인
- 배수구 막힘 확인

## FAQ

### Q: 관리를 못 하면 어떻게 되나요?
A: 자재에 따라 다릅니다. **목재**는 1년만 방치해도 심각한 손상이 발생할 수 있습니다. **합성목**은 2-3년 정도는 큰 문제 없지만 오염이 굳어집니다. **알루미늄**은 관리 없이도 수십 년 사용 가능합니다.

### Q: 비가 많이 오는데 괜찮나요?
A: 정상적으로 시공된 데크는 **비에 강합니다.** 다만 배수가 잘 되는지 확인하세요. 물이 고이면 문제가 생깁니다. 장마철에는 배수구 막힘만 확인하면 됩니다.

### Q: 겨울에 데크를 사용해도 되나요?
A: **사용 가능합니다.** 다만 결빙 시 미끄럼 주의가 필요합니다. 알루미늄과 합성목은 겨울 사용에 문제없고, 목재는 수분 흡수 후 결빙 시 균열 위험이 있으니 주의하세요.
