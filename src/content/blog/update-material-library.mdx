안녕하세요, 데크센터입니다. 많은 분들이 요청해주셨던 국내 인기 브랜드 자재를 대거 추가했습니다. 이제 총 **130종 이상**의 자재로 더욱 다양한 설계가 가능합니다.

## 신규 추가 브랜드

### 한솔데코 프리미엄 합성목 (8종)

국내 합성목 시장 점유율 1위 한솔데코의 프리미엄 라인이 추가되었습니다.

| 제품명 | 색상 | 특징 |
|--------|------|------|
| 우드앤 클래식 | 월넛, 티크, 오크 | 실제 목재 질감 구현 |
| 우드앤 모던 | 차콜, 그레이, 아이보리 | 매트한 현대적 느낌 |
| 우드앤 네이처 | 라이트오크, 다크브라운 | 자연 친화적 색감 |

**추천 용도**: 주거용, 카페 테라스, 펜션

### 삼화데크 알루미늄 시리즈 (5종)

가볍고 견고한 알루미늄 데크 전문 브랜드입니다.

| 제품명 | 마감 | 특징 |
|--------|------|------|
| AL-100 | 무광 실버 | 미니멀 디자인 |
| AL-200 | 우드 프린팅 | 목재 느낌 + 알루미늄 내구성 |
| AL-300 | 안트라싯 | 고급스러운 다크 톤 |
| AL-400 | 브론즈 | 따뜻한 금속 느낌 |
| AL-500 | 화이트 | 밝고 깨끗한 공간 |

**추천 용도**: 루프탑, 수영장 주변, 상업 시설

### 동양우드 친환경 목재 (7종)

FSC 인증 친환경 목재를 사용하는 브랜드입니다.

| 제품명 | 수종 | 특징 |
|--------|------|------|
| 에코파인 | 소나무 | 가성비 우수, 밝은 톤 |
| 에코오크 | 참나무 | 단단하고 내구성 좋음 |
| 에코월넛 | 호두나무 | 고급스러운 진한 색상 |
| 에코세다 | 삼나무 | 방부 효과, 은은한 향 |
| 에코레드 | 레드우드 | 붉은 톤, 병충해 강함 |
| 에코애쉬 | 물푸레 | 탄력 있고 무늬가 아름다움 |
| 에코티크 | 티크 | 최고급, 자연 방수 |

**추천 용도**: 전원주택, 조경, 친환경 건축물

### 타일마스터 타일형 데크 (10종)

조립식 타일 형태로 DIY 설치가 가능한 제품군입니다.

| 시리즈 | 재질 | 크기 | 특징 |
|--------|------|------|------|
| 이지타일 | WPC | 300×300mm | 클릭 조립, 초보자용 |
| 프로타일 | WPC+스톤 | 450×450mm | 돌 질감 표현 |
| 그래스타일 | WPC+인조잔디 | 300×300mm | 잔디 느낌 연출 |

**추천 용도**: 발코니, 베란다, 옥상 DIY

## 자재별 상세 정보 추가

### 실제 시공 사진

각 자재별로 실제 시공된 현장 사진을 추가했습니다. 3D 렌더링뿐 아니라 실제 적용된 모습을 확인하실 수 있습니다.

### 가격 정보 업데이트

2025년 12월 기준 최신 가격으로 업데이트되었습니다.

| 자재 유형 | 평균 단가 (평당) |
|-----------|-----------------|
| 합성목 | 15~25만원 |
| 알루미늄 | 25~40만원 |
| 천연 목재 | 12~30만원 |
| 타일형 | 8~15만원 |

> 가격은 참고용이며, 실제 견적은 시공 조건에 따라 달라질 수 있습니다.

### 내구성/유지보수 정보

각 자재의 특성을 한눈에 비교할 수 있도록 정보를 추가했습니다.

| 자재 | 예상 수명 | 유지보수 주기 | 관리 난이도 |
|------|----------|--------------|------------|
| 합성목 | 15~20년 | 연 1~2회 세척 | 쉬움 |
| 알루미늄 | 20년+ | 거의 불필요 | 매우 쉬움 |
| 천연 목재 | 10~15년 | 연 1~2회 오일 도포 | 보통 |
| 타일형 | 10~15년 | 필요시 교체 | 쉬움 |

## 자재 검색 기능 개선

### 새로운 필터 옵션

- **가격대별**: 저가형, 중가형, 고가형
- **용도별**: 주거용, 상업용, DIY
- **특성별**: 친환경, 내구성 우수, 유지보수 쉬움
- **색상별**: 밝은 톤, 어두운 톤, 그레이 톤

### 비교 기능

최대 3개 자재를 나란히 비교할 수 있는 기능이 추가되었습니다.

1. 비교할 자재 선택 (체크박스)
2. "비교하기" 버튼 클릭
3. 특성, 가격, 장단점 한눈에 비교

## Pro 회원 전용 자재

Pro 플랜 회원만 사용 가능한 프리미엄 자재 10종이 추가되었습니다.

### 프리미엄 전용 자재

- 해외 수입 브랜드 5종
- 특수 마감 처리 자재 3종
- 맞춤 제작 자재 시뮬레이션 2종

Pro 회원은 프리미엄 자재의 실제 샘플을 요청할 수 있는 서비스도 준비 중입니다. (2026년 상반기 출시 예정)

## 추가 희망 자재 접수

원하시는 자재가 라이브러리에 없다면 알려주세요!

### 요청 방법

1. 앱 내 **피드백** 버튼 클릭
2. "자재 추가 요청" 선택
3. 브랜드명, 제품명 입력
4. 가능하면 제품 페이지 링크 첨부

많은 분들이 요청해주신 자재는 우선적으로 추가됩니다.

### 최근 추가된 요청 자재

사용자 요청으로 이번에 추가된 자재:

- 한솔데코 우드앤 클래식 (15명 요청)
- 삼화데크 AL-300 안트라싯 (8명 요청)
- 타일마스터 그래스타일 (12명 요청)
