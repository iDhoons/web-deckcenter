기본 설계는 익숙해지셨나요? 이제 데크센터의 고급 기능을 활용해 프로페셔널한 설계를 완성해 보세요.

## 복잡한 형태 설계하기

직사각형 외에도 다양한 형태의 데크를 설계할 수 있습니다.

### L자형 데크

1. 형태 선택에서 **L자형**을 선택합니다
2. 각 변의 길이를 개별적으로 입력합니다
3. 모서리 부분의 마감 방식을 선택합니다
   - 직각 마감
   - 대각선 컷
   - 라운드 처리

### ㄷ자형 및 다각형

- **ㄷ자형**: 안뜰을 감싸는 형태에 적합
- **다각형**: 자유롭게 꼭지점을 추가하여 복잡한 형태 구현

### 자유형 설계

**자유형 모드**에서는 마우스로 직접 외곽선을 그릴 수 있습니다.

1. 툴바에서 **펜 도구** 선택
2. 클릭하여 꼭지점 추가
3. 더블클릭으로 형태 완성
4. 꼭지점을 드래그하여 미세 조정

## 계단 및 난간 추가하기

### 계단 추가

높이 차이가 있는 공간에는 계단이 필수입니다.

| 설정 | 옵션 |
|------|------|
| 계단 위치 | 데크 가장자리 선택 |
| 단수 | 자동 계산 또는 수동 입력 |
| 폭 | 600mm ~ 1200mm |
| 자재 | 데크와 동일 또는 별도 선택 |

> 계단은 건축법상 단높이 180mm 이하를 권장합니다. 앱에서 자동으로 적정 단수를 계산해 드립니다.

### 난간 설치

안전을 위해 난간이 필요한 경우:

1. **난간 추가** 버튼 클릭
2. 설치할 가장자리 선택 (복수 선택 가능)
3. 난간 스타일 선택
   - 수직 바
   - 수평 와이어
   - 유리 패널
4. 높이 및 간격 조정

## 다중 자재 조합 테크닉

하나의 데크에 여러 자재를 조합하면 더욱 개성 있는 디자인이 완성됩니다.

### 추천 조합

| 조합 | 효과 |
|------|------|
| 알루미늄 프레임 + 합성목 바닥 | 모던하면서 관리 편리 |
| 목재 바닥 + 돌 포인트 | 자연스러운 조경 느낌 |
| 다크 + 라이트 톤 교차 | 시각적 포인트 |

### 영역 분할 설계

1. **영역 분할** 도구 선택
2. 분할선을 그려 영역 구분
3. 각 영역에 다른 자재 적용

활용 예시:
- 바베큐 존과 휴식 존 구분
- 동선(통로)과 체류 공간 차별화
- 가구 배치 영역 시각화

## 실시간 비용 최적화

예산에 맞는 최적의 설계를 찾아보세요.

### 비용 최적화 기능

1. 상단 메뉴에서 **비용 최적화** 클릭
2. 목표 예산 입력
3. 시스템이 자동으로 대안 제시
   - 유사한 외관의 저렴한 자재
   - 형태 단순화 제안
   - 면적 조정 옵션

### 자재별 비용 비교

같은 면적에서 자재만 바꿔가며 실시간으로 비용 변화를 확인할 수 있습니다.

## 프레젠테이션 모드

고객에게 설계안을 보여줄 때 유용한 기능입니다.

### 프레젠테이션 모드 진입

- 단축키: `P`
- 또는 상단 메뉴 → **프레젠테이션**

### 프레젠테이션 기능

- **자동 회전**: 3D 뷰가 천천히 회전
- **주야간 전환**: 조명 시뮬레이션
- **치수 표시**: 주요 치수 오버레이
- **비교 모드**: 2개 시안 나란히 비교

## 템플릿 저장 및 재사용

자주 사용하는 설계를 템플릿으로 저장하면 효율적입니다.

### 템플릿 저장

1. 설계 완료 후 **템플릿으로 저장** 클릭
2. 템플릿 이름 및 카테고리 입력
3. 공개/비공개 설정

### 팀 공유 (Pro 플랜)

- 팀원과 템플릿 공유
- 회사 전용 자재 프리셋 생성
- 브랜드 가이드라인 적용
